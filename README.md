## 👋 Sobre mim

Construo software porque é a ferramenta mais direta que conheço pra transformar uma ideia em algo que existe no mundo. Antes de código, passei mais de uma década resolvendo problema real, na hora, sem manual — isso entrou na forma como penso sistema: pouca paciência pra solução bonita que não resolve nada, e gosto por arquitetura que se sustenta sob pressão.

Toda a produção pública aqui segue filosofia open source. Clona, lê, refatora, manda PR — código existe pra ser usado, não pra ficar trancado.
